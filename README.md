# muvv_rotas# 🌊 Muvv Rotas - Logística & Fintech

> **Status:** MVP Protótipo Funcional (High-Fidelity)  
> **Foco:** ZPE Piauí & Exportação Logística  

## 🚀 A Visão
A **Muvv Rotas** não é apenas um app de fretes. É o braço logístico da Holding Muvv, desenhado para reduzir o "atrito zero" entre o porto, a ZPE de Parnaíba e o transportador.

## 💡 Diferenciais de Negócio implementados:
1. **Taxas Dinâmicas:** Algoritmo de lucro baseado em categoria (Pesado 12%, Leve 8%, ZPE 15%).
2. **Ecossistema Bimodal:** Carteira digital integrada com conversão Real/Euro para operações portuárias.
3. **ZPE Gold Security:** Selo de prestígio para parceiros com apólice de seguro ativa.
4. **Interface Mares & Trilhas:** Design orgânico inspirado no litoral piauiense (Paleta Azure & Teal).

## 🛠️ Stack Técnica
- **Vite + React 18** (Velocidade total)
- **Tailwind CSS v4** (Estilização moderna e leve)
- **Framer Motion** (Animações de alta fidelidade)
- **SVG Mapping** (Rotas customizadas da BR-343 e Porto)
